# Customer Behavior Data Analysis

This project focuses on analyzing customer shopping behavior using data analytics techniques. 
The main objective is to understand purchasing patterns, customer preferences, and key trends 
that can help businesses make data-driven decisions.

The analysis is performed using **SQL, Python, and Power BI**. SQL is used for querying and 
extracting meaningful insights from the dataset, Python is used for data preprocessing and 
exploratory data analysis, and Power BI is used to create interactive dashboards for 
visualizing customer trends and business insights.

Through this project, various aspects of customer behavior are explored, such as purchasing 
frequency, product preferences, spending patterns, and the impact of discounts on customer 
purchases. The results are presented through visual reports and dashboards that help in 
understanding customer segments and overall shopping trends.

## Technologies Used
- SQL – Data querying and analysis
- Python – Data cleaning and exploratory data analysis
- Power BI – Interactive dashboards and data visualization

## Project Objectives
- Analyze customer shopping behavior
- Identify purchasing trends and patterns
- Understand the impact of discounts on sales
- Create interactive dashboards for business insights

## Project Files
- **Dataset (.csv)** – Customer shopping data used for analysis
- **SQL Queries (.sql)** – Queries used for extracting insights
- **Power BI Dashboard (.pbix)** – Interactive visualization dashboard
- **Analysis Documents (.pdf)** – Business problem and analytical insights

This project demonstrates how data analytics tools can be used to transform raw customer data 
into meaningful insights that support better business decision-making.
